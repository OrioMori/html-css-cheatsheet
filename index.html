Important Tags For Building a Webpage

Cards in this Stack:

Tag Description
Comments <!--...--> Defines a comment
<!DOCTYPE> Defines the document type
<html> Defines an HTML document
<head> Contains metadata/information for the document
<header> Defines a header for a document or section
<body> Defines the document's body
<footer> Defines a footer for a document or section

<-- Comments -->

Comments make clean code

Comments

help to maintain clean and readable code for when you add someone new to a project or need to go back to old code.

<!DOCTYPE html>

Defines an HTML Document.

<!DOCTYPE html>

Required by all browsers, this informationalInformational tags are NOT HTML tags and inform the browser about things like format or language. tag tells the browser that it is accessing an HTML document.

The declaration is NOT case sensitive.

Example:

<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
</head>

<body>
The content of the document......
</body>
</html>

<html>

The Root of an HTML Document

<html>

Contains all html code except for <!DOCTYPE> declaration.

Note: You should always include the lang attribute inside the tag, to declare the language of the Web page. This is meant to assist search engines and browsers.

<!DOCTYPE html>
<html lang="en">
<head>
<title>Title of the document</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

<head>

<head>

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
Metadata is data about the HTML document. Metadata is not displayed.
Metadata typically define the document title, character set, styles, scripts, and other meta information.

The following elements can go inside the <head> element:
<title> (Required by every webpage)
<base>
<style>
<link>
<meta>
<noscript>
<script>

<header>

<header>

represents a container for introductory content or a set of navigational links.

A <header> element typically contains:

One or more heading elements (<h1> - <h6>)
A logo or icon
Authorship information

Note: You can have several <header> elements in one HTML document. However, <header> cannot be placed within a <footer>, <address> or another <header> element.

Head Tags

Metadata and Linking

Cards in this Stack:

Tag Description
<title> Defines a title for the document
<base> Specifies the base URL/target for all relative URLs in a document
<style> Defines style information for a document
<link> Defines the relationship between a document and an external resource (most used to link to style sheets)
<meta> Defines metadata about an HTML document
<noscript> Defines an alternate content for users that do not support client-side scripts
<script> Defines a client-side script
